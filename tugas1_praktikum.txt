##3. Penjelasan singkat tentang perintah git:t : init, clone, remote, add, commit, push dan checkout
    - git init -> Membuat repository git di folder proyek
    - git clone -> Mengambil repository dari GitHub ke komputer lokal
    - git remote -> Menghubungkan repository lokal ke repository di GitHub
    - git add -> menambahkan file yang diubah ke daftar yang akan disimpan
    - git commit -> Menyimpan perubahan dengan pesan tertentu
    - git push -> Mengirim perubahan dari lokal ke repository di GitHub
    - git checkout -> Berpindah ke branch baru

##4. Jelaskan perbedaan antara GitHub dan GitLab
